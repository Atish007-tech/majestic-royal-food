# Majestic Royal Food 🍔✨🌒

A premium, cinematic food delivery platform built with **React** and **Node.js**, featuring a glassmorphic dark theme, 3D effects, and full mobile optimization.

## 🚀 How to Run Locally

To get this project running on your own machine, follow these steps:

### 1. Prerequisites
- **Node.js** (v16+)
- **Git**

### 2. Installation
Clone the repository and install all dependencies:

```bash
# Clone the repository
git clone https://github.com/Atish007-tech/majestic-royal-food.git

# Navigate to the project
cd majestic-royal-food

# Install all dependencies (Backend & Frontend)
npm run install-all
```

### 3. Run the App
Start both the backend and frontend simultaneously:

```bash
npm run dev
```
The app will be available at `http://localhost:3000`.

## 🌐 How to "Run using Link" (Deployment)

To make this website public (so others can visit a link like `https://your-site.render.com`), you need to **host** it. We recommend **[Render.com](https://render.com/)** for full-stack apps:

1.  **Sign in to Render** using your GitHub account.
2.  Click **"New +"** and select **"Web Service"**.
3.  Connect this repository (`Atish007-tech/majestic-royal-food`).
4.  **Settings**:
    - **Build Command**: `npm run install-all && npm run build --prefix frontend`
    - **Start Command**: `npm start --prefix backend` (Note: Ensure your backend serves the frontend build in production).
5.  Click **"Deploy"**!

Once finished, Render will provide you with a **Live URL**.
